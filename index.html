EXIficient XML becomes efficient ...

EXIficient is a set of open source implementations of the W3C Efficient XML Interchange (EXI) format specification (see projects on github.com/exificient) in various programming languages.

  • Java
  • JavaScript
  • C/C++

The EXI format is a very compact representation for the Extensible Markup Language (XML) Information Set that is intended to simultaneously optimize performance and the utilization of computational resources.
